灵魂序列
P1暂定 初窥灵魂
P2暂定 灵魂打击：攻击时，附带额外真实伤害，对无魂生物（has_soul == false）无效
P3暂定 灵魂具象：死亡后，转化为友方幽魂
P4暂定 灵魂方舟：收到致命伤害后，传送到所在村庄的篝火（无村庄或篝火损坏则继续死亡进程，可触发战斗续行）


咒法序列
P1无
P2基础咒法：允许在战斗时对敌人施加诅咒、中毒 等debuff
P3瘟疫行者：允许在战斗时对敌人施加瘟疫、灰化病 等debuff
P4血肉戏法：对敌人施加debuff时，扩大效果半径，额外造成伤害并延长持续时间
P5死亡低语：允许在战斗中对敌人施加死亡低语buff

腐化序列
P1无
P2探索腐化：身处腐化之地时，不会获得诅咒效果；不会在腐化之地死亡留下幽灵
P3掌控腐化：获得特质：暗影之根（原版），身处腐化之地获得增益buff、回复血量
P4腐化之源：可在战斗中通过法术扩散腐化之地，增益buff效果增强
P5死亡行军：增强扩散腐化之地法术
